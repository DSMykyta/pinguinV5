Модуль товарів

Повна система управління товарами, варіантами та групами. Плагінна архітектура, CRUD через Google Sheets, polling, wizard, SEO-генерація.

Архітектура

Модульна система: ядро (стан + дані + хуки) + плагіни (таблиці, модали, події). Будь-який плагін можна видалити без поломки ядра.

Принцип роботи

Точка входу main-products.js → initCore() → initProducts(). Плагіни завантажуються динамічно через import().

Крок 1
initCore()
Layout, nav, модали, тема, авторизація.
Крок 2
loadPlugins()
7 плагінів через Promise.allSettled. Кожен реєструє хуки.
Крок 3
checkAuthAndLoadData()
Паралельне завантаження: Products, Variants, Groups, Brands, Categories.
Крок 4
runHook('onInit')
Плагіни рендерять таблиці, ініціалізують event handlers.
