Таблиці

Генератор табличних даних для маркетплейсів.

Як використовувати:

  • Вибір маркетплейсу — оберіть потрібний маркетплейс у бічній панелі
  • Завантаження CSV — імпортуйте файл з товарами
  • Редагування — змінюйте дані прямо в таблиці
  • Експорт — збережіть результат у потрібному форматі

Гарячі клавіші:

  • Ctrl+C — копіювати виділене
  • Ctrl+V — вставити дані
  • Tab — перехід до наступної комірки

Редактор тексту

WYSIWYG редактор для створення та редагування текстів товарів з автоматичною перевіркою на заборонені слова та інтеграцією з SEO-генератором.

Як працює:

  • Вставте текст — скопіюйте текст товару в редактор
  • Автоматична перевірка — заборонені слова підсвічуються червоним
  • Інтеграція з SEO — система автоматично визначає бренд та назву товару з першого рядка тексту і заповнює поля в блоці SEO
  • Копіювання результату — Ctrl+C копіює HTML-код, Ctrl+Shift+C копіює чистий текст

Два режими роботи:

  • Текст — візуальний WYSIWYG редактор з форматуванням
  • Код — перегляд та редагування чистого HTML-коду

Панель інструментів:

  • B — жирний текст (<strong>)
  • I — курсив (<em>)
  • H1, H2, H3 — заголовки
  • Список — маркований список (<ul>)
  • всі маленькі — перетворення тексту на малі літери
  • ВСІ ВЕЛИКІ — перетворення тексту на великі літери
  • Кожне Слово — кожне слово з великої літери

Гарячі клавіші:

  • Ctrl+B — жирний текст
  • Ctrl+I — курсив
  • Ctrl+Z — скасувати (undo)
  • Ctrl+Y — повторити (redo)
  • Ctrl+C — копіювати як HTML
  • Ctrl+Shift+C — копіювати як текст
  • Enter — новий параграф
  • Shift+Enter — новий рядок (без параграфа)

Автовизначення бренду:

Система автоматично розпізнає бренд з першого рядка тексту. Бренд шукається у базі даних брендів (включаючи альтернативні назви). Після бренду йде назва товару — до першого тире.

Приклад: "Optimum Nutrition Gold Standard 100% Whey 2.27 кг — опис товару" → Бренд: "Optimum Nutrition", Товар: "Gold Standard 100% Whey 2.27 кг"

Статистика тексту:

  • Символів — загальна кількість символів
  • Слів — кількість слів у тексті
  • Час читання — приблизний час читання тексту

Підсвічування помилок:

  • Червоний — заборонені слова (наведіть для пояснення)
  • Жовтий — небезпечні HTML-патерни

SEO

Інструменти для оптимізації контенту під пошукові системи.

Функції:

  • Аналіз ключових слів — перевірка наявності ключових слів у тексті
  • Meta-теги — генерація title та description
  • Рекомендації — поради щодо покращення SEO

Бічна панель:

  • SEO-налаштування для різних маркетплейсів
  • Шаблони meta-тегів

Переклад

Інструмент для перекладу текстів між мовами.

Як використовувати:

  • Вибір мов — оберіть мову оригіналу та мову перекладу
  • Введення тексту — вставте текст для перекладу
  • Переклад — натисніть кнопку для отримання перекладу

Підтримувані мови:

  • Українська
  • Російська
  • Англійська

Офіційні сайти

Генератор посилань на офіційні сайти брендів.

Можливості:

  • Пошук бренду — знайдіть бренд за назвою
  • Генерація посилання — автоматичне створення HTML-посилання
  • Копіювання — швидке копіювання готового коду

Бічна панель:

  • Список брендів
  • Налаштування формату посилань

Обробка зображень

Інструмент для редагування та оптимізації зображень товарів.

Функції:

  • Зміна розміру — масштабування зображень під вимоги маркетплейсів
  • Обрізка — кадрування зображень
  • Оптимізація — зменшення розміру файлу без втрати якості
  • Водяний знак — додавання логотипу на зображення

Підтримувані формати:

  • JPG, PNG, WebP

Заборонені слова

Управління списком заборонених слів для перевірки текстів.

Можливості:

  • Додавання — додайте нові заборонені слова або фрази
  • Редагування — змініть існуючі записи
  • Видалення — видаліть непотрібні слова
  • Категорії — групуйте слова за типом (реклама, конкуренти, ненормативна лексика)

Поля запису:

  • Група — назва групи заборонених слів
  • Слова UA — українські варіанти (через кому)
  • Слова RU — російські варіанти (через кому)
  • Тип — тип заборони
  • Пояснення — чому слово заборонене
  • Підказка — чим замінити

Бренди

Управління базою даних брендів.

Можливості:

  • Додавання — додайте новий бренд
  • Редагування — змініть інформацію про бренд
  • Видалення — видаліть бренд з бази
  • Пошук — швидкий пошук бренду за назвою

Поля бренду:

  • Назва — основна назва бренду
  • Альтернативні назви — інші варіанти написання
  • Країна — країна походження
  • Опис — текст про бренд для глосарію
  • Сайт — офіційний сайт бренду

Ключові слова

Управління базою ключових слів та глосарієм.

Можливості:

  • Додавання — додайте нове ключове слово
  • Редагування — змініть інформацію про слово
  • Видалення — видаліть запис
  • Ієрархія — створюйте зв'язки між словами (батьківське слово)

Поля запису:

  • Тип — категорія ключового слова
  • Назва UA/RU/EN — переклади назви
  • Тригери — слова для автоматичного виявлення
  • Keywords UA/RU — SEO ключові слова
  • Текст глосарію — розгорнутий опис
