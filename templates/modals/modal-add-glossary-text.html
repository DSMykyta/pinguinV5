Додати опис терміну
Можна використовувати HTML теги: <p>, <ul>, <li>, <strong>, <code> тощо

Правила оформлення тексту глосарію

Структура тексту

  • Використовуйте <p> для параграфів
  • Використовуйте <ul> та <li> для списків
  • Виділяйте важливі терміни за допомогою <strong>
  • Код та технічні терміни огортайте в <code>

Приклад оформлення

<p>Білок - це складна органічна речовина, що складається з амінокислот.</p>

<p>Основні функції білків:</p>
<ul>
  <li>Будівельна функція</li>
  <li>Каталітична функція</li>
  <li>Захисна функція</li>
</ul>

<p>Приклад: альбумін, казеїн, колаген.</p>

Рекомендації

  • Пишіть коротко та зрозуміло
  • Починайте з визначення терміну
  • Додавайте приклади для кращого розуміння
  • Уникайте занадто складних конструкцій
