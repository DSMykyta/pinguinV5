Style Guide

Повний каталог дизайн-системи pinguin-v5. Кожен елемент, варіація, джерело та використання. 55 CSS файлів, 10 сторінок, 55 шаблонів.

Design Tokens

Централізовані змінні дизайн-системи. Визначені в root.css, підтримують light / dark через [data-theme="dark"]. Токени керують кольорами, відступами, радіусами та тінями всього інтерфейсу.

Primary / Secondary / Tertiary

Основна кольорова тріада бренду. Primary — акцентні кнопки, активні стани, навігація. Secondary — допоміжні елементи, hover. Tertiary — CTA-акценти. Кожен має пару: solid + on (текст поверху) + container (напівпрозорий фон).

--color-on-main#ffffff
--color-main-cContainer
--color-secondaryrgb(39, 93, 99)
--color-on-secondary#ffffff
--color-secondary-cContainer
--color-tertiaryrgb(232, 110, 80)
--color-on-tertiary#ffffff
--color-tertiary-cContainer

Status Colors

Семантичні кольори зворотного зв'язку. Кожен має пару: solid (іконки, текст) + container (фон chips). Error — валідація, видалення. Warning — попередження. Success — підтвердження. Info — підказки, нейтральні стани.

Surfaces

Шари поверхонь для створення глибини. Surface — базовий фон. Container low / mid / high — зростаючий контраст для вкладених елементів. On-surface — основний текст. On-surface-v — вторинний текст. Outline — розділювачі.

Buttons

Кнопки є основним інтерактивним елементом. button-base.css автоматично стилізує всі <button>.

.btn-primary

Основна кнопка дії. Брендовий фон, hover затемнює через filter: brightness(0.85). Одна на секцію — для головної дії (Зберегти, Додати, Створити).

.btn-secondary

Вторинна кнопка для менш важливих дій. Сірий фон --color-surface-c-high. Використовується поруч з primary або для окремих дій без акценту.

.btn-outline

Контурна кнопка — прозорий фон з бордером --color-outline. Більше уваги ніж secondary, менше ніж primary. Для альтернативних дій.

[pagination] charm

Декларативна пагінація. Додай атрибут pagination="N" на будь-який контейнер з дітьми — JS автоматично розріже їх на сторінки, створить навігацію (кнопки сторінок) та FAB (вибір кількості). MutationObserver слідкує за DOM — при ре-рендері таблиці пагінація оновлюється автоматично.

edit_note Editor

Універсальний WYSIWYG редактор з конфігурацією через HTML-чарми. Замість JS конфіг-об'єкта — декларативні атрибути на контейнері. Без чармів = readonly текстовий блок. Кожен чарм додає функціонал.

Система чармів

Кожен чарм — HTML-атрибут на контейнері редактора. Можна комбінувати в будь-якій комбінації.

editor-main.js editor-template.js
Чарм Що додає Плагіни
editor Базовий маркер. Без інших чармів — readonly блок (contenteditable="false") editor-paste, editor-undo, editor-enter
tools Панель інструментів: bold, italic, H1-H3, list, case + find/replace. Вмикає contenteditable="true" editor-formatting, editor-case, editor-find
code Перемикач Текст/Код (radio switch в тулбарі) editor-mode
check Перевірка заборонених слів: chips в тулбарі, підсвічування в тексті, tooltip editor-validation, editor-chip-navigation, editor-tooltip

cleanup-links Тогл очистки посилань у футері. Без значення = очистка ON. ="allow" = лінки дозволені editor-cleanup

JS ініціалізація

Фабрика читає чарми автоматично. В JS передаються тільки runtime-опції.

editor-main.js createHighlightEditor()
Базовий виклик (все з чармів)
import { createHighlightEditor } from '../common/editor/editor-main.js'; const container = document.getElementById('ghl-editor-container'); const editor = createHighlightEditor(container); // Чарми на контейнері визначають всю конфігурацію
З runtime-опціями
// initialValue та onChange — єдине що передається в JS const editor = createHighlightEditor(container, { initialValue: '<p>Початковий текст</p>', onChange: (html) => console.log('Changed:', html), });
Публічне API
editor.getValue(); // Чистий HTML (без підсвічувань) editor.setValue(html); // Встановити HTML editor.getPlainText(); // Тільки текст editor.getMode(); // 'text' або 'code' editor.setMode('code'); // Переключити режим editor.focus(); // Фокус на редактор editor.clear(); // Очистити все editor.destroy(); // Знищити екземпляр
